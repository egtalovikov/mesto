**Описание**

Одностраничная интерактивная социальная сеть с возможностью добавлять фотографии, удалять их, ставить лайки, редактирования профиля и просмотра увеличенного изображения.

**Используемые технологии**

* CSS Flexbox
* CSS Grid
* CSS Animations
* JavaScript
* Nested БЭМ
* Git
